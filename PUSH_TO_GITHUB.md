# Push to GitHub

The remote is set to **https://github.com/Alexander-Mitrofanov/MetagenomeGenerator**.

From your machine (where you’re logged into GitHub), run:

```bash
cd /home/alex/CursorProjects/MetagenomeGenerator
git push -u origin main
```

If you use SSH instead of HTTPS:

```bash
git remote set-url origin git@github.com:Alexander-Mitrofanov/MetagenomeGenerator.git
git push -u origin main
```
